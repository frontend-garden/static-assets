# Frontend Garden Static Assets

> Static assets for email, coding tools, etc.

## Content

Recommended structure:

- \<PLATFORM\>
  - \[\<PURPOSE\>\]
    - \[\<VERSION\>\]

No assets should be ever deleted once added so they remain available "forever".
